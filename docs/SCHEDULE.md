### Cat A 詳細サブタスク

#### A-1 環境構築・設計（4/14〜4/15）
- XcodeGen + project.yml 設定
- OpenCV xcframework 組み込み・Bridging Header 設定
- CoreML モデル選定・ダウンロード（XFeat / ALIKED）
- 座標系・データフロー設計

#### A-2 カメラ基盤実装（4/16〜4/21）
- ARSessionManager 実装（ARKit フレーム配信）
- CVPixelBuffer → UIImage 変換パイプライン
- scaledIntrinsics 実装（fx/fy/cx/cy の縮小比換算）
- SwiftUI カメラプレビュー（ARCameraView）

#### A-3 OpenCVブリッジ実装（4/22〜5/01）
- LK 光学フロー実装（calcOpticalFlowPyrLK）
- ORB 特徴点検出実装
- ArUco マーカー検出・姿勢推定
- Essential Matrix + recoverPose 実装
- 三角測量（triangulatePoints）実装

#### A-4 特徴点追跡エンジン（5/07〜5/18）
- TrackedPoint / TrackTier データ構造設計
- LK フロー毎フレーム追跡ループ実装
- 非同期再検出スケジューリング（NSLock でスレッドセーフ）
- pending 検出の既存トラックへの統合（mergePending）

#### A-5 CoreML記述子抽出（5/19〜5/28）
- XFeatMatcher 実装（グレースケール入力・64次元記述子）
- ALIKEDMatcher 実装（RGB入力・128次元記述子）
- NMS（Non-Maximum Suppression）実装

#### A-6 階層トラッカー統合（5/29〜6/05）
- ORB→XFeat→ALIKED 再検出間隔・バジェット設定
- BLAS cblas_sgemm によるコサイン類似度マッチング
- 統合動作確認・パフォーマンス計測

#### A-7 VOコア実装（6/08〜6/23）⚠️ 最難フェーズ
- VisualMap 実装（3D点群管理 + BLAS マッチング）
- 状態機械設計（searching / bootstrapping / tracking / lost）
- bootstrapping フェーズ実装（視差チェック + 三角測量）
- tracking フェーズ実装（findMatches → solvePnP RANSAC）
- growMap 実装（定期三角測量による新規点追加）

#### A-8 キーフレームDB + ロスト復帰（6/24〜7/03）
- StoredKeyframe データ構造設計
- 定期キーフレーム追加（keyframeInterval: 30f）
- サムネイル類似度ゲーティング実装
- tryDescriptorRecovery 実装（ロスト時の KF 全件試行）

#### A-9 ARオーバーレイ実装（7/06〜7/27）⚠️ クリティカルパス
- intrinsics → SCNCamera 投影行列変換実装
- VO ポーズ → SceneKit カメラ毎フレーム同期
- カメラ映像 + 3D シーン合成（SceneView 重畳）
- 重畳精度の検証と補正

#### A-10 3Dモデル配置（7/28〜8/03）
- USDZ モデルロード・シーンへの配置
- スケール・位置・回転の調整 UI

#### A-11 野外テスト＆調整（8/04〜8/25）

### 高リスク

| リスク | 影響 | 対策 |
|---|---|---|
| ARオーバーレイ（A-9）の投影行列ズレ | 重畳が大きくズレてデモにならない | 早期に実機で格子パターンを使って精度検証。1週遅れで即アウト |
